🏠 House Price Prediction App

This is an interactive Streamlit application that predicts house prices using machine learning models trained on the Kaggle House Prices Dataset.

🚀 Features

 manually input house features.

Predict house SalePrice instantly.

Uses Random Forest Regressor (with preprocessing for categorical & numerical data).

Shows evaluation metrics (RMSE & R²) on the sidebar.

Automatically handles missing features with imputation.
